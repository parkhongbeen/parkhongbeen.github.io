for문과 while문의 차이점

for문은 도와주는 변수(i)가 선언 후 for문이 끝나면 사라지는 반면, while문은 문법상 밖에서 선언하기 때문에 도와주는 변수(i)가 사라지지 않습니다. 물론 for문도 밖에서 도와주는 변수를 선언할 경우 for문이 끝나고 사라지지 않습니다.
