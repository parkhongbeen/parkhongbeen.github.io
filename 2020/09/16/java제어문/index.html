제어문(if, else if, else)

if문

조건은 true아니면 false 두가지 경우밖에 없습니다.
==, <, >, <=, >= 등의 부등호를 통해서 조건을 나타낼 수 있고, 바로 true, false 값을 줘서 조건을 표시 할 수도 있습니다.

else if, else

연산자

AND(&&), OR(||), NOT(!)

  • AND : 두 조건 모두 참
  • OR: 두 조건 중 하나라도 참
  • NOT: 부정
